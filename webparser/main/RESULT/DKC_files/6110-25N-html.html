Муфта труба-труба д.25мм, IP66/IP67, никелированная латунь

НАЗНАЧЕНИЕ И ОСОБЕННОСТИ


Муфта используется для соединения двух жестких металлических труб одного диаметра. Она обеспечивает высокую степень герметизации - IP 66-67. Материал аксессуара - никелированная латунь, - имеет высокую устойчивость к коррозии. Муфта совместима с трубами диаметром 25 мм.
